# 🔥 ForgeClaw Skills

> **Skill Extractor + Ultimate Harvester for OpenClaw Ecosystem**
>
> Extract, harvest, and manage 52+ verified skills and 50+ ecosystem tools from the OpenClaw community.

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/uncommonpope-png/forgeclaw-skills.git
cd forgeclaw-skills

# Run the extractor (52 original skills)
./run.sh

# Or run the ultimate harvester (50+ ecosystem tools)
node ultimate-harvester.js
```

## 📦 What's Included

### extractor.js
Extracts 52 verified skills from OpenClaw repository:
- Communication: Discord, Slack, WhatsApp, iMessage
- Productivity: Notion, Obsidian, Trello, Apple Notes
- Development: GitHub, coding-agent, tmux
- AI/ML: Gemini, OpenAI (image, whisper)
- Media: Spotify, Sonos, video-frames
- Utilities: weather, canvas, healthcheck

### ultimate-harvester.js
Harvests 50+ ecosystem tools from curated sources:
- **Memory Systems:** MemOS, MoltBrain, memU, memov, powermem
- **Alternative Architectures:** TinyClaw, IronClaw, NanoClaw, PicoClaw, MicroClaw, Carapace
- **MCP Servers:** openclaw-mcp-adapter, openclaw-claude-code-skill
- **Plugins:** WeChat, Feishu, DingTalk, QQ, HomeAssistant, Oura
